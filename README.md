
<h2>MVVM Android Example Project</h2>
<p>This project was created soley for the purpose of demonstrating coding experience. The app displays a list of books from a remote json file and includes a detail view when a book is selected.</p>
  <ul style="display: inline-block; text-align: left;">
    <li>Offline first approach with RoomDB as local data holder</li>
    <li>DI with Hilt</li>
    <li>Jetpack Compose with State management</li>
    <li>Kotlin</li>
    <li>MVVM</li>
  </ul>
